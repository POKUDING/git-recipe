#비빔밥

1. 고슬고슬한 밥을 짓는다.
2. 콩나물을 무친다.
3. 시금치 나물을 무친다.
4. 소고기를 볶는다.
5. 계란후라이를 굽는다.
6. 밥위에 같은 나물의 색깔이 이웃하지 않도록 올린다.
